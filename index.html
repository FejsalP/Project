User manual

Part 1 (calculator): user has to enter 2 binary numbers (i.e. only 1 and 0 are allowed) and then click one of the operations +, - , * or / representing addition, subtraction, multiplication and division, respectively. User can change between 4, 8, 16, 32 and 64 bits. Note: changing bits will restart inputs.
Part 2.1 (conversion to IEEE-754 notation): user has to enter a decimal number and then click Convert button next to the input. User can change between 32 and 64 bits. Note: changing bits will restart input (will afect input in part 2.2).
Part 2.2 (conversion from IEEE-754 notation): user has to enter 3 parts of the IEEE-754 notation (signbit, biased exponent and mantissa) and then click Convert button next to the input. User can change between 32 ad 64 bits. Note: changing bits will restart input (will affect input in part 2.1).
Part 3 (conversion between bases): user has to select from and to which base they want to change, and also input number they would like to convert. Changing the value of the base from which user wants to change number will reset the input.

This is part of the project for the class CS304 Computer Architecture at the International University of Sarajevo.
